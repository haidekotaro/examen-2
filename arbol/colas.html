Colas en Java

¿Qué es una Cola?

En Java, una cola es una estructura de datos FIFO (First In, First Out), donde el primer elemento en entrar es el primero en salir. La interfaz Queue en Java define métodos clave para manejar colas add(e): Agrega un elemento (lanza excepción si está llena). offer(e): Similar a add, pero devuelve false si falla. poll(): Retira y devuelve el primer elemento, o null si está vacía. peek(): Muestra el primer elemento sin retirarlo, o null si está vacía.

Implementaciones Comunes

                 LinkedList: Cola básica, permite inserciones y eliminaciones rápidas.
             PriorityQueue: Cola de prioridad, ordena elementos en lugar de usar FIFO.
             ArrayDeque: Cola eficiente, admite inserciones y eliminaciones en ambos extremos.

    Ejemplo rapido

                Queuestring queue = new LinkedList<>();
            queue.add("A");
            queue.offer("B");
            System.out.println(queue.poll()); // Retira y muestra "A"
            System.out.println(queue.peek()); // Muestra "B" sin retirarlo
