Pilas en Java

¿Qué es una Pila?

Una pila es una estructura de datos que sigue el principio LIFO (Last In, First Out), lo que significa que el último elemento agregado es el primero en ser retirado.

Operaciones Principales

  • push: Agrega un elemento a la cima de la pila.
  • pop: Elimina y retorna el elemento en la cima de la pila.
  • peek: Retorna el elemento en la cima de la pila sin eliminarlo.
  • isEmpty: Verifica si la pila está vacía.

Implementación de Pilas en Java

En Java, se puede utilizar la clase Stack de la biblioteca estándar para trabajar con pilas. Aquí hay un ejemplo simple:

import java.util.Stack;

public class EjemploPila {
    public static void main(String[] args) {
        Stack pila = new Stack<>();
        
        // Agregar elementos
        pila.push(1);
        pila.push(2);
        pila.push(3);
        
        // Mostrar el elemento en la cima
        System.out.println("Elemento en la cima: " + pila.peek());
        
        // Retirar el elemento
        System.out.println("Elemento retirado: " + pila.pop());
        
        // Verificar si la pila está vacía
        System.out.println("¿La pila está vacía? " + pila.isEmpty());
    }
}
